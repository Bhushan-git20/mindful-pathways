# MindMate - Student Wellness Platform

A comprehensive mental health and wellness platform designed for students, featuring self-assessments, journaling, chatbot support, wellness tracking, and curated resources.

## Features

- **Self-Assessment**: Take PHQ-9 or GAD-7 questionnaires to check in with yourself
- **Journal Entry**: Write about your thoughts and feelings
- **Chatbot Support**: Get guidance from an AI wellness assistant
- **View Insights**: See your wellness progress over time
- **Resources**: Access mental health resources and support
- **Wellness Streak**: Track your daily check-in streak

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:8080`.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn/UI
- Supabase (Auth & Database)
- Recharts (Data Visualization)
